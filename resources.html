See how font size units impact text size*

Figuring out how unit type changed font sizing on webpages was one of the more difficult parts of learning web design for me, all those years ago. To help others see how text unit types work, I made a seperate, stand alone website that you can visit here.

Otherwise, use the input boxes below to see how number and unit type affects my name! Keep in mind that the standard root size for most browsers is 16px, unless manually changed, but the parent size of the changable text is 20px, so you can see how EM and REM are different.
