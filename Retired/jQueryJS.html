jQuery has incredible uses. When a web site does not require data exchanges, as this website does not, jQuery or JavaScript are one of the few ways for your website to make decisions. Even when creating web applications or websites that allow the transferance of data, jQuery or JavaScript is still nearly always used. Although the color change menu is entirely CSS, the ability to change color is simple jQuery: 
$('#palette1').click(function(){
	$('.primary').css({"color":"#1995AD"});
	$('.contrast').css({"color":"#8c8b8c;"});
	$('.complementary').css({"color":"#A1D6E2;"});
	$('.mellow').css({"color":"#F1F1F2"});

	$('.bgPrimary').css({"background-color":"#1995AD"});
	$('.bgcontrast').css({"background-color":"#8c8b8c;"});
	$('.bgComplementary').css({"background-color":"#A1D6E2;"});
	$('.bgMellow').css({"background-color":"#F1F1F2"});
});
This incredibly simple code allows the user to select the background colors they prefer simply by changing the styling of different html classes. JQuery can also be used to create and pass JSON objects, place formatting masks on input groups, hide and unhide elements on a page, activate and deactivate buttons, and so much more. Through building countless web application pages and simple web sites, Ren is very familiar with the capabilities of JQuery.
